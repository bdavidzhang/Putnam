Problem 1. Find all n ∈ ℤ+ such that there exist solutions to

2an + 3bn = 4cn,

where a,b,c ∈ ℤ+.

Problem 2. Find all polynomials p(x) such that there exist polynomials q(x) that satisfy the equation:

p(p(x)) - x = (p(x) - x)2q(x).

Problem 3. Let S be the set of bijections

T : {1,2,3}×{1,2,…,2024}→{1,2,…,6072} (1)

such that T(1,j) < T(2,j) < T(3,j) for all j ∈{1,2,…,2024} and T(i,j) < T(i,j + 1) for all i ∈{1,2,3} and j ∈{1,2,…,2023}. Do there exist a and c in {1,2,3} and b and d in {1,2,…,2024} such that the fraction of elements in T in S for which T(a,b) < T(c,d) is at least 1∕3 and at most 2∕3?

Problem 4. Find all primes p > 5 for which there exists an integer a and an integer r satisfying 1 ≤ r ≤ p - 1 with the following property: the sequence 1,a,a2,…,ap-5 can be arranged to form a sequence b0,b1,…,bp-5 such that bn -bn-1 -r is divisible by p for 1 ≤ n ≤ p - 5.

Problem 5. Let Ω be a circle of radius 9 centered at the origin. Let Δ be a disk of radius 1 with center (r,0), r ∈ [0,8]. For two points P,Q on Ω, find the r that minimizes the probability for PQ to be in Δ.

Problem 6. For some sufficiently small x, write

= ∑ k=0∞c kxk.

For some n ∈ ℤn, let A be the n×n matrix with i,j entry ci+j-1 for i,j ∈{1,…,n}. Find the determinant of A.
